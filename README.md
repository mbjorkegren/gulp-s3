# gulp-s3 [![NPM version][npm-image]][npm-url]

> s3 plugin for [gulp](https://github.com/wearefractal/gulp)

## Usage

First, install `gulp-s3` as a development dependency:

```shell
npm install --save-dev gulp-s3
```

Setup your aws.json file
```javascript
{
  "key": "AKIAI3Z7CUAFHG53DMJA",
  "secret": "acYxWRu5RRa6CwzQuhdXEfTpbQA+1XQJ7Z1bGTCx",
  "bucket": "dev.example.com",
  "region": "eu-west-1"
}
```

Then, use it in your `gulpfile.js`:
```javascript
var s3 = require("gulp-s3");

aws = JSON.parse(fs.readFileSync('./aws.json'));
options = {
    delay: 1000, // optional delay each request by x milliseconds
    makeUploadPath: function(file) {
        return file.path.match(RegExp("dist.*"))[0];
    },
};
gulp.src('./dist/**', {read: false})
    .pipe(s3(aws, options));
```
